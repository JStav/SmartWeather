Temperature

The Smart Weather API returns temperature values in degrees Celcius (°C). Use the formula below to convert the returned temperature values to to Farenheit (°F).

$$T_{f} = (\frac{9}{5})\times{}T_{c}+32$$

Source: http://www.crh.noaa.gov/Image/epz/wxcalc/tempConvert.pdf
