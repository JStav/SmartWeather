Wind Chill

$$WindChill = 35.74 + (0.6215\times{} T) - \Big(35.75\times{}Wind_{sfc}^{0.16}\Big) + \Big(0.4275\times{}T\times{}Wind_{sfc}^{0.16}\Big)$$

\(T\) = degrees Fahrenheit (°F)

\(WindSpeed_{sfc}\) = mph

Source: https://www.weather.gov/media/epz/wxcalc/windChill.pdf
