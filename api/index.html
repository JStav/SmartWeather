Overview

This API gives access to observation and metadata for WeatherFlow Smart Weather Stations. You can learn more about WeatherFlow Smart Weather at http://www.weatherflow.com/smart-home-weather-stations/ . To see an example of what you can do with this API visit http://smartweather.weatherflow.com

Object Model

Security

Smart Weather Station users have the option of setting the data for each of their stations as either public or private. The API has two access levels (Public and User).

  • Public Access Level with API Key
    • Access to all Stations currently set as "public"
    • Send your API Key as "api_key" in the query string with all API requests
  • User Access Level with Oauth2 Access Token
    • Access to all data owned by the user's access token
    • Access to all Stations currently set as "public"
    • Send the access token as "token" in the query string with all API requests
    • Oauth2 Implicit Flow
      • Use this flow to get an access token that never expires. This is the token you send with all API requests.
