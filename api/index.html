Overview

This API gives access to observation and metadata for WeatherFlow Smart Weather Stations. You can learn more about WeatherFlow Smart Weather at http://www.weatherflow.com/smart-home-weather-stations/ . To see an example of what you can do with this API visit http://smartweather.weatherflow.com

Object Model

Security

Smart Weather Station users have the option of setting the data for each of their stations as either public or private. The API has two access levels (Public and User).
